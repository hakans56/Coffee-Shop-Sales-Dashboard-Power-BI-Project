Project Overview

This project is a Sales Performance Dashboard built in Power BI using a public Kaggle dataset. The dashboard simulates a coffee shop business scenario and provides actionable insights for decision-makers.
The main goal was to create a clean, executive-level dashboard that is both visually appealing and analytically robust, enabling quick performance monitoring and strategic planning.

📊 Key Metrics
Total Revenue
Total Orders
Total Customers
Average Order Value
Customer Satisfaction Score
Month-over-Month Growth (%)

📈 Analysis Highlights

Monthly Revenue Trend
Revenue Breakdown by Product Category
Regional Performance Comparison
Customer Segment (Gender) Analysis
This dashboard enables quick identification of high-performing regions/products, customer satisfaction trends, and growth opportunities.

🎨 Design Features

Clean, minimalistic layout for clarity
KPI cards at the top for instant insights
Color-coded charts to highlight trends and outliers
Interactive filters for region, product type, and time period

🛠 Tools Used
Power BI Desktop
Kaggle Public Dataset
Basic data cleaning and transformation within Power BI

🚀 Key Takeaways

Demonstrates end-to-end dashboard development
Highlights ability to transform raw data into actionable insights
Showcases visual design, KPI tracking, and analytical thinking
